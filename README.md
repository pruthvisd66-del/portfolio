# SD Pruthvi Gowda - Personal Portfolio

A responsive personal portfolio website showcasing my skills, projects, and experience as a Full Stack Developer and BCA student at KLE SNC.

## Overview

This portfolio website serves as my digital resume, highlighting my journey in full-stack development, open-source contributions, and academic pursuits. Built with modern web technologies, it provides a clean and professional presentation of my work and achievements.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Dark/Light Mode Toggle**: User preference saved in localStorage
- **Smooth Scrolling Navigation**: Seamless navigation between sections
- **Project Showcase**: Detailed cards for each project with links to GitHub and live demos
- **Contact Form**: Simple form with basic validation
- **Fast Loading**: Optimized for performance with minimal assets

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with responsive design
- **Interactivity**: Vanilla JavaScript for smooth scrolling, dark mode, and form validation
- **Deployment**: GitHub Pages

## Sections

1. **Hero**: Introduction with name, role, and summary
2. **About**: Background and current learning focus
3. **Skills**: Categorized skills (Frontend, Backend, Tools, Concepts)
4. **Projects**: Showcase of 3+ projects with descriptions and links
5. **Experience**: Internship details and achievements
6. **Education**: Academic background
7. **Contact**: Contact information and form

## How to Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/sdpruthvi/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your browser:
   - Double-click the file, or
   - Use a local server (e.g., `python -m http.server` or VS Code Live Server extension)

## Deployment

This website is deployed on GitHub Pages. To deploy your own version:

1. Fork this repository
2. Customize the content in `index.html`
3. Go to repository Settings > Pages
4. Select "Deploy from a branch" and choose "main" branch
5. Your site will be live at `https://yourusername.github.io/portfolio`

## Customization

- Update personal information in `index.html`
- Modify styles in `styles.css`
- Add more interactivity in `script.js`
- Replace placeholder images in `assets/` folder with real ones
